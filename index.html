Как указать размер флекс-элемента?

Cвойство flex-basis задает размер на главной оси.

Имеет те же значения, что и width / height.
Если не указано, то базовый размер возьмется из ширины / высоты содержимого.

Flex

flex-basis: 50%

flex-basis: 50%

flex-basis: auto

flex-basis: 50%

flex-basis: auto

Flex

Почему размер базовый?

Это исходный размер. Свободное место можно распределять в соответствии с коэффициентом жадности flex-элемента - flex-grow.

Flex

flex-grow: 0 // По-умолчанию flex-grow: 0 1 flex-grow: 0 2
flex-grow: 0

Flex

Что, если сумма базовых размеров больше чем свободного места?

Будем делить отрицательное пространство в соответствии с коэффициентами сжатия - flex-shrink.

Flex

flex-shrink: 1 // По-умолчанию flex-shrink: 0 1 flex-shrink: 0 2
Сила упругости, возникающая в теле при его деформации, прямо пропорциональна величине этой деформации.

Алгоритм расчета

Flex

flex: [flex-grow] [flex-shrink] [flex-basis];

                .block__element {
                    flex: initial; // = 0 1 auto — все по умолчанию
                    flex: auto; // = 1 1 auto
                    flex: none; // = 0 0 auto
                    flex: 2; // число -> flex-grow -> = 2 1 auto
                    flex: 50%; // единица измерения длины -> flex-basis = 50% -> = 0 1 50%
                    flex: 2 50%; // flex-grow & flex-basis -> = 2 1 50%
                    flex: 2 0 50%;
                }

Flex

Особенности внешних отступов у элементов во flex-контейнере:

  • не схлопываются
  • не выпадают
  • отступ с auto заберет все свободное место по своему направлению
